# Welcome to Opencode Installation!


# Steps to install Opencode 


### 1. Open a new tab in browser 
search for "https://opencode.ai/" 
copy command "npm i -g opencode-ai" or "curl -fsSL https://opencode.ai/install | bash"
and run it in terminal/shell and once installed run command "opencode"

### 2. Now open "https://github.com/code-yeongyu/oh-my-opencode" 

now paste the command "curl -s https://raw.githubusercontent.com/code-yeongyu/oh-my-opencode/refs/heads/master/docs/guide/installation.md" into the input section of Opencode terminal

**Enjoy!**
